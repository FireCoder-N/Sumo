# Sumo
An experimental repo documenting the process of converting an Open Street Map to Unity.

This repo follows the tutorial ["How to Create a Network and Simulation in SUMO Using OpenStreetMap"](https://www.youtube.com/watch?v=wZycufsTEGU) of [Dr. Joanne Skiles](https://github.com/chaotictoejam) as presented at their [YouTube channel](https://www.youtube.com/@DrJoanneSkiles). The commands shown in the video are presented here for ease of access, each in its own file, alongside links to their respective documentation.

Later on, I have experimented with converting the SUMO simulation into a Unity scene, using the [SUMO2Unity repository](https://github.com/SimuTraffX-Lab/SUMO2Unity) by [SimuTraffX Lab](https://github.com/SimuTraffX-Lab) and the video-tutorials from their [RoadWayVR](https://www.youtube.com/@RoadwayVR) YouTube channel.

## Pipeline
Export Open Street Map as .osm file -> netconvert -> randomTrips -> (duarouter)
